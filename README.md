## 세팅 방법

1. /src/main/webapp/WEB-INF/mybatis/jdbc.properties 파일 생성

```
jdbc.driverClass=net.sf.log4jdbc.DriverSpy
jdbc.url=jdbc:log4jdbc:oracle:thin:@아이피:1521:xe
jdbc.username=계정명
jdbc.password=비밀번호
```
